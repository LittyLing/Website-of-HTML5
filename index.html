What is the Website of HTML5?

The Website of HTML5 is a modern HTML5, CSS3, Javascript, and JQuery based website that allows users to discover the structure of HTML5 websites as well as fundamentals of modern web design.

Our Goal

The Website of HTML5 wishes to help users uncover how modern HTML5 websites are structured with new HTML5 elements and features such as section, article, aside, etc. In addition, the Website of HTML5 hopes to allow users to learn modern web design fundamentals for their future experiences when developing and designing websites. Users can learn from the website's multiple pages, observe and copy the website code as a template, and visit external links listed on the website.
